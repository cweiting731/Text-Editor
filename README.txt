** 存檔規則說明

1. 若是剛開啟檔案或是剛新增一個檔案，且並未對內容做變更的話，按下「結束」或是「刪除視窗」會默認已存檔，直接退出

2. 自動存檔只對開啟舊檔有效，新增的檔案不會自動存檔

3. 若是新增檔案後有儲存過一次 (確認了存檔位置)，自動存檔功能便會開啟

4. 測試 auto save 時如果覺得等待太久，可以去程式碼把 AUTO_SAVE_INTERVAL 調低一點

5. 開啟舊檔與存檔預設位置會在專案資料夾中 saves 裡面

6. 開啟舊檔的視窗標題不會是檔案路徑，只會取檔案名字而已，新增檔案則一律為 newPage。如果新增檔案已經儲存，則視窗標題會變為檔案的名字

** 字數統計規則說明

字數統計會忽略空白字元

** undo redo 規則說明

undo redo 的上限為 20 筆，只要有更改 text 就會被記錄，更改字型與顏色一樣會被記錄
